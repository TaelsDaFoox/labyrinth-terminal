# LABYRINTH TERMINAL
spooky little game made for Hack Club Siege.

SOFTWARE USED:

ENGINE - Godot

3D ASSETS - Blender

2D ASSETS - Piskel, Paint.net

AUDIO - jsfxr, Audacity

ADDITIONAL ASSET USED: https://sketchfab.com/3d-models/crt-monitor-e2dd2887a8904e4fa3d5a32e2935adb9

play the game at https://taelsdafoox.itch.io/labyrinth-terminal

it's a simple game of managing energy to defend yourself from weird spooky things in a way that controls like Pac-Man.
